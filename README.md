# Tesla Prize Portal System

## Professional Fan Registration & Rewards Platform

A comprehensive web application for managing Tesla fan registrations, displaying prizes, tracking winnings, and providing customer support.

### Features

- **Fan Registration System**: Secure registration and login for Tesla fans
- **Admin Dashboard**: Manage selected fans and assign prizes
- **Fan Dashboard**: View personal information, winnings, and rewards
- **Prize Management**: Input and track prize information
- **Support System**: Two-way support ticketing system for fans and admin
- **Professional UI/UX**: Modern, responsive design

### Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT
